## 使用要求

可以通过两种方式来运行代理池，一种方式是使用 Docker（推荐），另一种方式是常规方式运行，要求如下：

### Docker

如果使用 Docker，则需要安装如下环境：

* Docker
* Docker-Compose

安装方法自行搜索即可。

### 常规方式

常规方式要求有 Python 环境、Redis 环境，具体要求如下：

* Python>=3.6
* Redis

## Docker 运行

如果安装好了 Docker 和 Docker-Compose，只需要一条命令即可运行。

```shell script
docker-compose up
```
